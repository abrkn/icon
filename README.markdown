icon
===

Convert an image to thumbnails suitable for favicon, metro tiles, touch icons.

Install
---

```
npm install -g icon
```

Usage
---

```
icon original.png
ls | select name

Name
----
favicon.ico
favicon.png
original.png
touch-icon-ipad-retina.png
touch-icon-iphone-retina.png
touch-icon-iphone.png
tileicon.png
```
